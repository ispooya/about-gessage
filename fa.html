پیام‌های کامیت گیت با هوش مصنوعی

پیام‌های کامیت با کیفیت بالا از تغییرات staged با استفاده از چندین بک‌اند هوش مصنوعی تولید کنید. چندپلتفرمی، امن و دوستدار توسعه‌دهنده.

شروع کنید نمایش دمو
$ curl -fsSL https://raw.githubusercontent.com/ispooya/gessage-cli/refs/heads/main/install.sh | bash
✨ Installing Gessage CLI...
🎉 Installation complete!
$ gessage setup --model openrouter
🔧 Setting up OpenRouter (Free)...
$ git add . && gessage
🤖 Generated: feat: add user authentication system

چرا Gessage را انتخاب کنیم؟

قابلیت‌های قدرتمند طراحی شده برای گردش کار توسعه مدرن

چندین بک‌اند هوش مصنوعی

پشتیبانی از OpenRouter (رایگان)، OpenAI GPT-4o و مدل‌های محلی Ollama. بهترین هوش مصنوعی را برای نیازتان انتخاب کنید.

امنیت اول

حذف خودکار اطلاعات محرمانه تضمین می‌کند که داده‌های حساس شما هرگز دستگاهتان را ترک نکرده یا به ارائه‌دهندگان هوش مصنوعی نرسد.

کامیت‌های متعارف

اجباری کردن فرمت درست پیام کامیت با عنوان ≤ ۷۲ کاراکتر، متن ≤ ۱۰۰ ستون و انواع استاندارد.

انتخاب خودکار هوشمند

به طور خودکار بهترین مدل را بر اساس اندازه diff انتخاب می‌کند، با گزینه‌های تنظیم دستی در دسترس.

جریان تعاملی

تایید، ویرایش، تولید مجدد یا لغو پیام‌های کامیت قبل از اعمال. کنترل کامل بر روی کامیت‌هایتان.

چندپلتفرمی

بدون مشکل در لینوکس، macOS و ویندوز کار می‌کند. نصب یک خطی برای همه پلتفرم‌ها.

مدل‌های هوش مصنوعی پشتیبانی شده

از چندین ارائه‌دهنده هوش مصنوعی انتخاب کنید تا با گردش کار و بودجه‌تان مطابقت داشته باشد

Gessage را در عمل ببینید

نمونه گردش کار واقعی

ببینید که چگونه Gessage تغییرات staged شما را تحلیل می‌کند و پیام‌های کامیت کاملی مطابق با استانداردهای Conventional Commits تولید می‌کند. دیگر کامیت‌هایی مثل "fix stuff" یا "update code" نخواهید داشت!

$ git add src/auth.js
$ gessage
🔍 Analyzing staged changes...
🤖 Generating commit message...
✨ Generated message:
feat(auth): implement JWT token validation
✅ Commit created successfully!

git add .
